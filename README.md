# Xubuntu-Modded
If you want install customize Xubuntu, just open Termux one click!

```bash
pkg install wget && wget https://github.com/ubuntu-termux-by-lukeyt/Xubuntu-Modded/raw/refs/heads/main/commands/install-xubuntu-modded.sh && chmod +x install-xubuntu-modded.sh && ./install-xubuntu-modded.sh
```
