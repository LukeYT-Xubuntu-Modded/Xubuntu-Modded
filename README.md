# Xubuntu-Modded

```bash
pkg install wget && wget https://github.com/LukeYT-Xubuntu-Modded/Xubuntu-Modded/blob/main/commands/install-xubuntu-modded.sh && chmod +x install-xubuntu-modded.sh && ./install-xubuntu-modded.sh
```
